## Arduino

- [Arduino IDE](https://www.arduino.cc/en/Main/Software)
- Hardware files for ESP8266: [Arduino for ESP8266](https://github.com/esp8266/Arduino)

### Libraries

- HTTP server: [ESPAsyncWebServer](https://github.com/me-no-dev/ESPAsyncWebServer)
- Generic library for low power wireless hardware (including RFM69): [RadioHead](http://www.airspayce.com/mikem/arduino/RadioHead/)
- Initial approach for Linked IoT bridge: [esp-radiohead-gateway](https://github.com/bergos/esp-radiohead-gateway)
- MQTT client for Arduino: [MQTT client](https://github.com/knolleary/pubsubclient)
- WiFi Connection manager: [WiFiManager](https://github.com/tzapu/WiFiManager)
